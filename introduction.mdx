## What is Rescuers API?

Rescuers API is a REST API for managing animal rescue operations, caregivers, and lost/found animal listings. Built with TypeScript and Express.js, it provides a comprehensive backend solution for animal rescue organizations to coordinate their activities, track animals, and connect with caregivers.

## Technology Stack

The API is built with modern, production-ready technologies:

- **Runtime**: Node.js with TypeScript
- **Framework**: Express.js
- **Database**: Firebase Firestore
- **Storage**: Firebase Storage
- **Authentication**: JWT tokens with bcrypt password hashing
- **Email**: Nodemailer for transactional emails
- **Image Processing**: Sharp for image optimization
