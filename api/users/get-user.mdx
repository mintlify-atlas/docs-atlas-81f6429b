## Get All Users

<Note>
This endpoint does not require authentication.
</Note>

Retrieve a list of all users in the system.

### Endpoint

```
GET /api/auth/
```

### Request

No parameters required.

### Response

<ResponseField name="data" type="array">
  Array of user objects
  
  <Expandable title="User Object">
    <ResponseField name="id" type="string">
      Unique identifier for the user
    </ResponseField>
    
    <ResponseField name="name" type="string">
      User's first name
    </ResponseField>
    
    <ResponseField name="lastName" type="string">
      User's last name
    </ResponseField>
    
    <ResponseField name="email" type="string">
      User's email address
    </ResponseField>
    
    <ResponseField name="image" type="string">
      URL to user's profile image
    </ResponseField>
    
    <ResponseField name="contacts" type="array">
      Array of contact information objects
    </ResponseField>
    
    <ResponseField name="roles" type="array">
      Array of role IDs
    </ResponseField>
    
    <ResponseField name="emailConfirmed" type="boolean">
      Email verification status
    </ResponseField>
    
    <ResponseField name="zoneId" type="string">
      Associated zone ID
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl --request GET \
  --url https://api.rescuers.com/api/auth/
```

```javascript JavaScript
const response = await fetch('https://api.rescuers.com/api/auth/', {
  method: 'GET'
});

const users = await response.json();
console.log(users);
```

```python Python
import requests

response = requests.get('https://api.rescuers.com/api/auth/')
users = response.json()
print(users)
```
</CodeGroup>

### Example Response

```json
[
  {
    "id": "user_123abc",
    "name": "John",
    "lastName": "Doe",
    "email": "john.doe@example.com",
    "image": "https://storage.example.com/users/profile.jpg",
    "contacts": [
      {
        "contact": "+1234567890",
        "type": "phone"
      }
    ],
    "roles": [1, 2],
    "emailConfirmed": true,
    "emailVerificationAttempts": 1,
    "zoneId": "zone_456def"
  }
]
```

## Get User by ID

<Note>
This endpoint does not require authentication.
</Note>

Retrieve a specific user by their unique identifier.

### Endpoint

```
GET /api/auth/:id
```

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the user to retrieve
</ParamField>

### Response

<ResponseField name="id" type="string">
  Unique identifier for the user
</ResponseField>

<ResponseField name="name" type="string">
  User's first name
</ResponseField>

<ResponseField name="lastName" type="string">
  User's last name
</ResponseField>

<ResponseField name="email" type="string">
  User's email address
</ResponseField>

<ResponseField name="image" type="string">
  URL to user's profile image
</ResponseField>

<ResponseField name="contacts" type="array">
  Array of contact information objects
  
  <Expandable title="Contact Object">
    <ResponseField name="contact" type="string">
      Contact value
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Contact type (e.g., "phone", "mobile", "email")
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="roles" type="array">
  Array of role IDs assigned to the user
</ResponseField>

<ResponseField name="emailConfirmed" type="boolean">
  Whether the user's email has been verified
</ResponseField>

<ResponseField name="emailVerificationAttempts" type="number">
  Number of email verification attempts
</ResponseField>

<ResponseField name="zoneId" type="string">
  ID of the zone associated with the user
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl --request GET \
  --url https://api.rescuers.com/api/auth/user_123abc
```

```javascript JavaScript
const userId = 'user_123abc';
const response = await fetch(`https://api.rescuers.com/api/auth/${userId}`, {
  method: 'GET'
});

const user = await response.json();
console.log(user);
```

```python Python
import requests

user_id = 'user_123abc'
response = requests.get(f'https://api.rescuers.com/api/auth/{user_id}')
user = response.json()
print(user)
```
</CodeGroup>

### Example Response

```json
{
  "id": "user_123abc",
  "name": "John",
  "lastName": "Doe",
  "email": "john.doe@example.com",
  "image": "https://storage.example.com/users/profile.jpg",
  "contacts": [
    {
      "contact": "+1234567890",
      "type": "phone"
    },
    {
      "contact": "john.work@example.com",
      "type": "email"
    }
  ],
  "roles": [1, 2],
  "emailConfirmed": true,
  "emailVerificationAttempts": 1,
  "zoneId": "zone_456def"

```

### Error Responses

<ResponseField name="message" type="string">
  Error message
</ResponseField>

<ResponseField name="code" type="number">
  HTTP status code
</ResponseField>

```json Error Response (404)

  "code": 500

```

## Get User Roles

<Note>
This endpoint requires authentication. Include a valid JWT token in the Authorization header.
</Note>

Retrieve the roles assigned to the authenticated user.

### Endpoint

```
GET /api/auth/roles
```

### Authentication

Requires a valid JWT token in the Authorization header:

```
Authorization: Bearer YOUR_JWT_TOKEN
```

### Response

<ResponseField name="roles" type="array">
  Array of role IDs assigned to the authenticated user
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl --request GET \
  --url https://api.rescuers.com/api/auth/roles \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://api.rescuers.com/api/auth/roles', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});

const roles = await response.json();
console.log(roles);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
}

response = requests.get('https://api.rescuers.com/api/auth/roles', headers=headers)
roles = response.json()
print(roles)
```
</CodeGroup>

### Example Response

```json
[1, 2, 5]
```

### Error Responses

```json Unauthorized (401)
{
  "message": "Unauthorized",
  "code": 401
}
```

```json Internal Server Error (500)
{
  "message": "Error interno del servidor.",
  "code": 500
}
```
