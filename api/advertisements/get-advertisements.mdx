## Get All Advertisements

<api method="GET" path="/api/advertisements">
  Retrieve a list of all advertisements.
</api>

### Headers

No authentication required.

### Response

<ResponseField name="advertisements" type="array">
  Array of advertisement objects
  <Expandable title="Advertisement Object">
    <ResponseField name="id" type="string">
      Unique identifier
    </ResponseField>
    <ResponseField name="userId" type="string">
      User who created the advertisement
    </ResponseField>
    <ResponseField name="content" type="string">
      Advertisement content
    </ResponseField>
    <ResponseField name="publicationDate" type="string">
      Publication date (ISO 8601)
    </ResponseField>
    <ResponseField name="state" type="number">
      Advertisement state
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl https://your-api-domain.com/api/advertisements
```

```javascript JavaScript
const response = await fetch('https://your-api-domain.com/api/advertisements');
const data = await response.json();
console.log(data.advertisements);
```

```python Python
import requests

response = requests.get('https://your-api-domain.com/api/advertisements')
data = response.json()
print(data['advertisements'])
```
</CodeGroup>

### Example Response

```json
{
  "advertisements": [
    {
      "id": "adv123",
      "userId": "user456",
      "content": "Adopt a rescue dog today!",
      "publicationDate": "2024-01-15T10:30:00Z",
      "state": 1
    }
  ]
}
```

## Get Advertisement by ID

<api method="GET" path="/api/advertisements/:advertisementId">
  Retrieve a specific advertisement by its ID.
</api>

### Path Parameters

<ParamField path="advertisementId" type="string" required>
  The unique identifier of the advertisement
</ParamField>

### Response

<ResponseField name="advertisement" type="object">
  The requested advertisement object
  <Expandable title="Properties">
    <ResponseField name="id" type="string">
      Unique identifier
    </ResponseField>
    <ResponseField name="userId" type="string">
      User who created the advertisement
    </ResponseField>
    <ResponseField name="content" type="string">
      Advertisement content
    </ResponseField>
    <ResponseField name="publicationDate" type="string">
      Publication date (ISO 8601)
    </ResponseField>
    <ResponseField name="state" type="number">
      Advertisement state
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl https://your-api-domain.com/api/advertisements/adv123
```

```javascript JavaScript
const advertisementId = 'adv123';
const response = await fetch(`https://your-api-domain.com/api/advertisements/${advertisementId}`);
const data = await response.json();
console.log(data.advertisement);
```

```python Python
import requests

advertisement_id = 'adv123'
response = requests.get(f'https://your-api-domain.com/api/advertisements/{advertisement_id}')
data = response.json()
print(data['advertisement'])
```
</CodeGroup>

### Example Response

```json
{
  "advertisement": {
    "id": "adv123",
    "userId": "user456",
    "content": "Adopt a rescue dog today!",
    "publicationDate": "2024-01-15T10:30:00Z",
    "state": 1
  }
}
```
