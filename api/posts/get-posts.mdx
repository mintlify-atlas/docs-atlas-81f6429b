## Overview

Retrieve a filtered list of posts from the Rescuers platform. This endpoint supports advanced filtering by state, attributes, pagination, and ownership.

<Note>
  This endpoint uses POST method to support complex filter payloads, despite being a read operation.
</Note>

## Endpoint

```
POST /api/posts
```

## Authentication

**Optional** - Authentication is only required when using the `onlyOwnerPublished` filter.

Include your Bearer token in the Authorization header:

```bash
Authorization: Bearer YOUR_ACCESS_TOKEN
```

## Request Body

<ParamField body="pageIndex" type="number" required>
  Zero-based page index for pagination. Start with 0 for the first page.
  
  **Example:** `0`
</ParamField>

<ParamField body="pageSize" type="number" required>
  Number of posts to return per page. Recommended values: 10-50.
  
  **Example:** `20`
</ParamField>

<ParamField body="atributes" type="string[]" required>
  Array of attribute IDs to filter posts by. Attributes can include species, size, color, age, etc. Pass an empty array `[]` to skip attribute filtering.
  
  **Example:** `["attr-dog-001", "attr-size-medium"]`
</ParamField>

<ParamField body="state" type="PostStates" required>
  Filter posts by their current state:
  
  - `1` - **Pending Review**: Waiting for editor review
  - `2` - **Draft**: Not ready for publication
  - `3` - **Published**: Publicly visible posts
  - `4` - **Archived**: Archived posts
  - `5` - **Rejected**: Rejected by editors
  - `6` - **Deleted**: Deleted posts
  
  **Example:** `3` (for published posts)
</ParamField>

<ParamField body="lost" type="boolean" required>
  Filter for lost vs found animals:
  
  - `true` - Show lost animal posts
  - `false` - Show found animal posts
  
  **Example:** `false`
</ParamField>

<ParamField body="onlyOwnerPublished" type="boolean" required>
  Control ownership filtering:
  
  - `true` - Return only posts owned by the authenticated user (requires authentication)
  - `false` - Return all posts matching other criteria
  
  **Example:** `false`
</ParamField>

## Response

Returns an object containing the filtered posts array and total count.

<ResponseField name="posts" type="Post[]">
  Array of post objects matching the filter criteria.
  
  <Expandable title="Post Object Properties">
    <ResponseField name="id" type="any">
      Unique identifier for the post
    </ResponseField>
    
    <ResponseField name="title" type="string">
      The post title
    </ResponseField>
    
    <ResponseField name="contentType" type="number">
      Type of content:
      - `1` - Animal
      - `2` - Advertisement
    </ResponseField>
    
    <ResponseField name="state" type="number">
      Current post state (1-6)
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Detailed description of the post
    </ResponseField>
    
    <ResponseField name="image" type="string">
      URL or path to the post's main image
    </ResponseField>
    
    <ResponseField name="postCategory" type="string">
      Category classification (e.g., "dogs", "cats", "birds")
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="totalCount" type="number">
  Total number of posts matching the filter criteria across all pages. Use this for pagination UI.
</ResponseField>

```javascript JavaScript
const filterPosts = async () => {
  const response = await fetch('https://api.rescuers.example/api/posts', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      pageIndex: 0,
      pageSize: 10,
      atributes: [],
      state: 3,
      lost: false,
      onlyOwnerPublished: false
    })
  });
  
  const data = await response.json();
  
  console.log(`Found ${data.totalCount} posts`);
  console.log('Posts:', data.posts);
  
  return data;
};

filterPosts();
```

```python Python
import requests

def get_posts(page=0, page_size=10, state=3):
    url = "https://api.rescuers.example/api/posts"
    
    payload = {
        "pageIndex": page,
        "pageSize": page_size,
        "atributes": [],
        "state": state,
        "lost": False,
        "onlyOwnerPublished": False
    }
    
    response = requests.post(url, json=payload)
    response.raise_for_status()
    
    data = response.json()
    
    print(f"Total posts: {data['totalCount']}")
    print(f"Returned: {len(data['posts'])} posts")
    
    return data

result = get_posts(page=0, page_size=20)
```

```typescript TypeScript
interface FilterRequest {
  pageIndex: number;
  pageSize: number;
  atributes: string[];
  state: number;
  lost: boolean;
  onlyOwnerPublished: boolean;
}

interface Post {
  id: any;
  title: string;
  contentType: number;
  state: number;
  description?: string;
  image?: string;
  postCategory?: string;
}

interface FilterResponse {
  posts: Post[];
  totalCount: number;
}

const filterPosts = async (
  filter: FilterRequest
): Promise<FilterResponse> => {
  const response = await fetch('https://api.rescuers.example/api/posts', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify(filter),
  });
  
  if (!response.ok) {
    throw new Error('Failed to fetch posts');
  }
  
  return response.json();
};

// Usage
const result = await filterPosts({
  pageIndex: 0,
  pageSize: 10,
  atributes: [],
  state: 3,
  lost: false,
  onlyOwnerPublished: false,
});
```

## Pagination Example

To implement pagination, use the `totalCount` and `pageSize` to calculate the number of pages:

```javascript
const totalPages = Math.ceil(data.totalCount / filter.pageSize);

// Fetch next page
filter.pageIndex = 1;
const nextPage = await filterPosts(filter);
```

## Notes

<Note>
  The `onlyOwnerPublished` filter only takes effect when the user is authenticated. If set to `true` without authentication, it will be ignored and all matching posts will be returned.
</Note>

<Note>
  When filtering by attributes, the endpoint returns posts that match ALL specified attributes (AND logic, not OR).
</Note>
