## Overview

Zones represent geographic or organizational areas with support for hierarchical parent-child relationships. Use zones to organize resources by location, region, or administrative boundaries.

## Zone Object

The Zone object contains the following fields:

<ResponseField name="id" type="string">
  Unique identifier for the zone
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the zone
</ResponseField>

<ResponseField name="code" type="string" required>
  Unique code identifier for the zone
</ResponseField>

<ResponseField name="zoneType" type="string" required>
  Type classification of the zone (e.g., country, state, city, district)
</ResponseField>

<ResponseField name="parentZoneId" type="string">
  ID of the parent zone for hierarchical organization (optional)
</ResponseField>

## Get All Zones

<api>GET /api/zones</api>

Retrieves a list of all zones in the system.

### Authentication

No authentication required

### Response

<ResponseField name="data" type="array">
  Array of zone objects
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET https://api.rescuers.example/api/zones \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.rescuers.example/api/zones', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});
const zones = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.rescuers.example/api/zones',
    headers={'Content-Type': 'application/json'}
)
zones = response.json()
```
</CodeGroup>

### Example Response

```json
{
  "data": [
    {
      "id": "zone123",
      "name": "North District",
      "code": "ND-001",
      "zoneType": "district",
      "parentZoneId": "city456"
    },
    {
      "id": "city456",
      "name": "Metro City",
      "code": "MC-001",
      "zoneType": "city"
    }
  ]
}
```

## Create Zone

<api>POST /api/zones</api>

Creates a new zone in the system.

### Authentication

**Required:** Bearer token authentication

Include the authentication token in the Authorization header:
```
Authorization: Bearer YOUR_TOKEN_HERE
```

### Request Body

<ParamField body="name" type="string" required>
  Display name of the zone
</ParamField>

<ParamField body="code" type="string" required>
  Unique code identifier for the zone
</ParamField>

<ParamField body="zoneType" type="string" required>
  Type classification of the zone
</ParamField>

<ParamField body="parentZoneId" type="string">
  ID of the parent zone (optional)
</ParamField>

### Response

<ResponseField name="data" type="object">
  The newly created zone object with generated ID
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST https://api.rescuers.example/api/zones \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -d '{
    "name": "East District",
    "code": "ED-001",
    "zoneType": "district",
    "parentZoneId": "city456"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.rescuers.example/api/zones', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN_HERE'
  },
  body: JSON.stringify({
    name: 'East District',
    code: 'ED-001',
    zoneType: 'district',
    parentZoneId: 'city456'
  })
});
const newZone = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.rescuers.example/api/zones',
    headers={
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_TOKEN_HERE'
    },
    json={
        'name': 'East District',
        'code': 'ED-001',
        'zoneType': 'district',
        'parentZoneId': 'city456'
    }
)
new_zone = response.json()
```
</CodeGroup>

### Example Response

```json
{
  "data": {
    "id": "zone789",
    "name": "East District",
    "code": "ED-001",
    "zoneType": "district",
    "parentZoneId": "city456"
  }
}
```

<Note>
  Status code: `201 Created`
</Note>

## Update Zone

<api>PUT /api/zones</api>

Updates an existing zone.

### Authentication

**Required:** Bearer token authentication

Include the authentication token in the Authorization header:
```
Authorization: Bearer YOUR_TOKEN_HERE
```

### Request Body

<ParamField body="id" type="string" required>
  ID of the zone to update
</ParamField>

<ParamField body="name" type="string">
  Updated display name
</ParamField>

<ParamField body="code" type="string">
  Updated code identifier
</ParamField>

<ParamField body="zoneType" type="string">
  Updated zone type
</ParamField>

<ParamField body="parentZoneId" type="string">
  Updated parent zone ID
</ParamField>

<Note>
  Include the `id` field and any fields you want to update. Omitted fields will remain unchanged.
</Note>

### Response

<ResponseField name="data" type="object">
  The updated zone object
</ResponseField>

<CodeGroup>
```bash cURL
curl -X PUT https://api.rescuers.example/api/zones \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -d '{
    "id": "zone123",
    "name": "North District (Updated)",
    "code": "ND-001"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.rescuers.example/api/zones', {
```

## Delete Zone

<api>DELETE /api/zones/:id</api>

Deletes a zone from the system.

### Path Parameters

<ParamField path="id" type="string" required>
  ID of the zone to delete
</ParamField>

### Authentication

**Required:** Bearer token authentication

Include the authentication token in the Authorization header:
```
Authorization: Bearer YOUR_TOKEN_HERE
```

### Response

Returns a success status with no content body.

<CodeGroup>
```bash cURL
curl -X DELETE https://api.rescuers.example/api/zones/zone123 \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

```javascript JavaScript
const zoneId = 'zone123';
const response = await fetch(`https://api.rescuers.example/api/zones/${zoneId}`, {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN_HERE'
  }
});
```

```python Python
import requests

zone_id = 'zone123'
response = requests.delete(
    f'https://api.rescuers.example/api/zones/{zone_id}',
    headers={'Authorization': 'Bearer YOUR_TOKEN_HERE'}
)
```
</CodeGroup>

<Note>
  Status code: `200 OK` on successful deletion
</Note>

## Error Responses

All endpoints may return the following error responses:

<Expandable title="400 Bad Request">
  Invalid request parameters or malformed request body.
  
  ```json
  {
    "error": "Invalid request data"
  }
  ```
</Expandable>

<Expandable title="401 Unauthorized">
  Missing or invalid authentication token (for protected endpoints).
  
  ```json
  {
    "error": "Authentication required"
  }
  ```
</Expandable>

<Expandable title="404 Not Found">
  The requested zone does not exist.
  
  ```json
  {
    "error": "Zone not found"
  }
  ```
</Expandable>

<Expandable title="500 Internal Server Error">
  Server-side error occurred.
  
  ```json
  {
    "error": "Error interno del servidor."
  }
  ```
</Expandable>
