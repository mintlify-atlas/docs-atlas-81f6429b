---
title: 'Create Animal'
api: 'POST /api/animals'
description: 'Create a new animal rescue post'
---

## Authentication

<Note>
  This endpoint requires authentication. Include a valid Bearer token in the Authorization header.
</Note>

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

## Request Body

<ParamField body="name" type="string" required>
  The name of the animal
</ParamField>

<ParamField body="description" type="string">
  Detailed description of the animal, including any relevant information about its condition, personality, or special needs
</ParamField>

<ParamField body="state" type="number" required>
  Current state of the post:
  - `1` - Pending Review
  - `2` - Draft
  - `3` - Published
  - `4` - Archived
  - `5` - Rejected
  - `6` - Deleted
</ParamField>

<ParamField body="lost" type="boolean" required>
  Indicates whether the animal is lost (`true`) or available for adoption/rescue (`false`)
</ParamField>

<ParamField body="atributes" type="string[]" required>
  Array of attributes describing the animal (e.g., ["Dog", "Golden Retriever", "3 years old", "Friendly"])
</ParamField>

<ParamField body="image" type="string">
  URL of the animal's image (optional, can be added later via upload endpoint)
</ParamField>

<Note>
  The `userId` field is automatically set from the authenticated user's JWT token and should not be included in the request body.
</Note>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the newly created animal post
</ResponseField>

<ResponseField name="name" type="string">
  Name of the animal
</ResponseField>

<ResponseField name="userId" type="string">
  ID of the user who created the post
</ResponseField>

<ResponseField name="image" type="string">
  URL of the animal's image (if provided)
</ResponseField>

<ResponseField name="description" type="string">
  Description of the animal
</ResponseField>

<ResponseField name="state" type="number">
  Current state of the post
</ResponseField>

<ResponseField name="lost" type="boolean">
  Whether the animal is lost
</ResponseField>

<ResponseField name="atributes" type="string[]">
  Array of animal attributes
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST https://api.rescuers.com/api/animals \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -d '{
    "name": "Max",
    "description": "Friendly golden retriever looking for a loving home. Great with kids and other pets.",
    "state": 3,
    "lost": false,
    "atributes": ["Dog", "Golden Retriever", "3 years old", "Male", "Vaccinated"]
  }'
```

```javascript JavaScript
const response = await fetch('https://api.rescuers.com/api/animals', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  },
  body: JSON.stringify({
    name: 'Max',
    description: 'Friendly golden retriever looking for a loving home. Great with kids and other pets.',
    state: 3,
    lost: false,
    atributes: ['Dog', 'Golden Retriever', '3 years old', 'Male', 'Vaccinated']
  })
});

const data = await response.json();
```

```python Python
import requests

url = 'https://api.rescuers.com/api/animals'
headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
}
data = {
    'name': 'Max',
    'description': 'Friendly golden retriever looking for a loving home. Great with kids and other pets.',
    'state': 3,
    'lost': False,
    'atributes': ['Dog', 'Golden Retriever', '3 years old', 'Male', 'Vaccinated']
}

response = requests.post(url, json=data, headers=headers)
result = response.json()
```

</CodeGroup>

## Error Codes

| Status Code | Description |
|-------------|-------------|
| 201 | Animal post created successfully |
| 400 | Bad request - invalid or missing required fields |
| 401 | Unauthorized - missing or invalid authentication token |
